# Bank Account Management System

A console-based Bank Account Management System developed using Java.

## 📌 Project Overview
This project simulates basic banking operations using Object-Oriented Programming principles.

## 🚀 Features
- Create Bank Account
- Deposit Money
- Withdraw Money
- Check Account Balance
- Display Account Details
- Input Validation
- Exception Handling

## 🛠 Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class

## 💡 Concepts Implemented
- Encapsulation
- Constructors
- Methods
- Control Structures (Switch, Loops)
- Input Validation

## ▶ How to Run

```bash
javac *.java
java BankManagementSystem
